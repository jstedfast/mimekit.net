	string fileName,
	bool append = true
)

Parameters

fileName
Type: SystemString
The file name.
append (Optional)
Type: SystemBoolean
true if the file should be appended to; otherwise, false. Defaults to true.
Remarks
Creates a new ProtocolLogger to log to a specified file. The file is created if it does not exist.
