Remarks

Loads a MimeMessage from the given stream, using the specified ParserOptions.

If persistent is and stream is seekable, then the MimeParser will not copy the content of MimeParts into memory. Instead, it will use a BoundStream to reference a substream of stream. This has the potential to not only save memory usage, but also improve MimeParser performance.
