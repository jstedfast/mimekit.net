Remarks
This default implementation for ApplicationPgpEncrypted nodes calls VisitApplicationPgpEncrypted(ApplicationPgpEncrypted). Override this method to call into a more specific method on a derived visitor class of the MimeVisitor class. However, it should still support unknown visitors by calling VisitApplicationPgpEncrypted(ApplicationPgpEncrypted).
