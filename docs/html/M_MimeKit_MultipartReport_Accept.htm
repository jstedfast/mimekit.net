Remarks
This default implementation for MultipartReport nodes calls VisitMultipartReport(MultipartReport). Override this method to call into a more specific method on a derived visitor class of the MimeVisitor class. However, it should still support unknown visitors by calling VisitMultipartReport(MultipartReport).
