Remarks

Maps the DigestAlgorithm to the appropriate string identifier as used by the micalg parameter value of a multipart/signed Content-Type header. For example:

AlgorithmName
MD2md2
MD4md4
MD5md5
Sha1sha-1
Sha224sha-224
Sha256sha-256
Sha384sha-384
Sha512sha-512
Tiger192tiger-192
RipeMD160ripemd160
Haval5160haval-5-160
