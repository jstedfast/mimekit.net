Remarks

The UUEncoding is an encoding that predates MIME and was used to encode binary content such as images and other types of multimedia to ensure that the data remained intact when sent via 7bit transports such as SMTP.

These days, the UUEncoding has largely been deprecated in favour of the base64 encoding, however, some older mail clients still use it.
