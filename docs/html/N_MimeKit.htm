MimeKit Namespace

The MimeKit namespace provides classes that are used to implement the core MIME parsing services of the framework.
Classes
 ClassDescription
AttachmentCollection A collection of attachments.
BodyBuilder A message body builder.
ContentDisposition A class representing a Content-Disposition header value.
ContentType A class representing a Content-Type header value.
DomainList A domain list.
ExperimentalMimeParser An experimental replacement for MimeParser.
FormatOptions Format options for serializing various MimeKit objects.
GroupAddress An address group, as specified by rfc0822.
Header A class representing a Message or MIME header.
HeaderIdExtensionsHeaderId extension methods.
HeaderList A list of Headers.
HeaderListCollection A collection of HeaderList groups.
InternetAddress An abstract internet address, as specified by rfc0822.
InternetAddressList A list of email addresses.
MailboxAddress A mailbox address, as specified by rfc822.
MessageDeliveryStatus A message delivery status MIME part.
MessageDispositionNotification A message disposition notification MIME part.
MessageFeedbackReport A message feedback report MIME part.
MessageIdList A list of Message-Ids.
MessagePart A MIME part containing a MimeMessage as its content.
MessagePartial A MIME part containing a partial message as its content.
MimeContent Encapsulates a content stream used by MimePart.
MimeEntity An abstract MIME entity.
MimeEntityBeginEventArgs Event args emitted by the MimeParser when it begins parsing a MimeEntity.
MimeEntityConstructorArgs MIME entity constructor arguments.
MimeEntityEndEventArgs Event args emitted by the MimeParser when a MimeEntity is parsed.
MimeIterator An iterator for a MIME tree structure.
MimeMessage A MIME message.
MimeMessageBeginEventArgs Event args emitted by the MimeParser when it begins parsing a MimeMessage.
MimeMessageEndEventArgs Event args emitted by the MimeParser when a MimeMessage is parsed.
MimeParser A MIME message and entity parser.
MimePart A leaf-node MIME part that contains content such as the message body text or an attachment.
MimeReader A MIME message and entity reader.
MimeTypes A mapping of file name extensions to the corresponding MIME-type.
MimeVisitor Represents a visitor for MIME trees.
Multipart A multipart MIME entity which may contain a collection of MIME entities.
MultipartAlternative A multipart/alternative MIME entity.
MultipartRelated A multipart/related MIME entity.
MultipartReport A multipart/report MIME entity.
Parameter A header parameter as found in the Content-Type and Content-Disposition headers.
ParameterList A list of parameters, as found in the Content-Type and Content-Disposition headers.
ParseException A Parse exception as thrown by the various Parse methods in MimeKit.
ParserOptions Parser options as used by MimeParser as well as various Parse and TryParse methods in MimeKit.
TextPart A textual MIME part.
TextRfc822Headers A MIME part containing message headers as its content.
