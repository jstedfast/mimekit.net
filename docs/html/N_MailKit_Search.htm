MailKit.Search Namespace

The MailKit.Search namespace provides classes that are necessary for searching folders for messages matching a set of criteria.
Classes
 ClassDescription
AnnotationSearchQuery An annotation-based search query.
BinarySearchQuery A binary search query such as an AND or OR expression.
DateSearchQuery A date-based search query.
FilterSearchQuery A filter-based search query.
HeaderSearchQuery A header-based search query.
NumericSearchQuery A numeric search query.
OrderBy Specifies a sort order for search results.
OrderByAnnotation Specifies an annotation-based sort order for search results.
SearchQuery A specialized query for searching messages in a IMailFolder.
SearchResults The results of a search.
TextSearchQuery A text-based search query.
UidSearchQuery A unique identifier-based search query.
UnarySearchQuery A unary search query such as a NOT expression.
Enumerations
 EnumerationDescription
OrderByType The field to sort by.
SearchOptions Advanced search options.
SearchTerm A search term.
SortOrder An enumeration of sort orders.
